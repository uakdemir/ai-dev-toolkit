# Contract Test Templates

Structural test code templates for barrel-file contract enforcement. Loaded at
Step 9 (Generate Artifacts) after the user approves modules in Phase 2.

Each template is a complete, runnable test file for its target stack. One test
file is generated per guarded module. Placeholders are substituted by the
orchestrator using confirmed module data from the Module Analysis Format.

## Placeholder Inventory

| Placeholder | Source | Example |
|---|---|---|
| `{MODULE_NAME}` | Module name | `auth` |
| `{MODULE_PATH}` | Module directory path | `src/auth` |
| `{BARREL_PATH}` | Barrel file path | `src/auth/index.ts` |
| `{SOURCE_EXTENSIONS}` | File extensions to scan | `['.ts', '.js']` |
| `{MODULE_INTERNAL_DIRS}` | Dirs within the module (everything except the barrel) | `['src/auth/strategies', 'src/auth/guards']` |
| `{WORKSPACE_PACKAGES}` | Package name → directory map (monorepo only, `{}` for single-project) | `{ '@myorg/auth': 'packages/auth' }` |
| `{MODULE_PASCAL}` | PascalCase module name (.NET only) | `Auth` |
| `{BARREL_NAMESPACE}` | Public barrel namespace (.NET only) | `MyApp.Auth` |
| `{MODULE_INTERNAL_NAMESPACES}` | Internal sub-namespaces (.NET only) | `["MyApp.Auth.Strategies"]` |

All placeholders are filled from the per-module analysis collected during
Phase 1 (codebase scan) and confirmed during Phase 2 (user review).

## Dedup Rules

- **One file per module.** File naming encodes the module name, so re-runs
  simply overwrite the existing file for that module.
- No start/end markers needed; the entire file is replaced on re-generation.
- The marker comment at the top of each generated file identifies the source
  skill and module for traceability.

## Node.js (Jest/Vitest) Template

File: `tests/structural/api-contracts-{MODULE_NAME}.test.ts`

```typescript
import * as fs from 'fs';
import * as path from 'path';
import { glob } from 'glob';

// --- Generated by api-contract-guard: {MODULE_NAME} ---

const MODULE_NAME = '{MODULE_NAME}';
const MODULE_PATH = '{MODULE_PATH}';
const BARREL_PATH = '{BARREL_PATH}';
const SOURCE_EXTENSIONS = [{SOURCE_EXTENSIONS}];
// WORKSPACE_PACKAGES maps package names to directories (monorepo only, empty for single-project).
// Populated by the orchestrator from workspace config. E.g.: { '@myorg/auth': 'packages/auth' }
const WORKSPACE_PACKAGES: Record<string, string> = {WORKSPACE_PACKAGES};

/**
 * Collect all source files in the project OUTSIDE the guarded module directory.
 */
function getConsumerFiles(): string[] {
  const patterns = SOURCE_EXTENSIONS.map(ext => `**/*${ext}`);
  const allFiles = patterns.flatMap(p => glob.sync(p, { ignore: ['node_modules/**'] }));
  const moduleDirResolved = path.resolve(MODULE_PATH);
  return allFiles.filter(f => !path.resolve(f).startsWith(moduleDirResolved + path.sep)
    && path.resolve(f) !== moduleDirResolved);
}

/**
 * Extract import/require statements from a source file.
 * Returns an array of { line, importPath } for each static import or require call.
 */
function extractImports(filePath: string): Array<{ line: number; importPath: string }> {
  const src = fs.readFileSync(filePath, 'utf-8');
  const lines = src.split('\n');
  const results: Array<{ line: number; importPath: string }> = [];

  for (let i = 0; i < lines.length; i++) {
    const text = lines[i];

    // ES import: import ... from 'path'  or  import 'path'
    const esMatch = text.match(/(?:import\s+.*?\s+from\s+|import\s+)['"]([^'"]+)['"]/);
    if (esMatch) {
      results.push({ line: i + 1, importPath: esMatch[1] });
      continue;
    }

    // CommonJS require: require('path')
    const cjsMatch = text.match(/require\s*\(\s*['"]([^'"]+)['"]\s*\)/);
    if (cjsMatch) {
      results.push({ line: i + 1, importPath: cjsMatch[1] });
    }
  }

  return results;
}

/**
 * Check whether an import path resolves to a location inside the guarded module.
 *
 * Handles two cases:
 * 1. Relative imports (./foo, ../foo) — resolved against consumer file directory
 * 2. Bare specifier imports (@myorg/auth/service) — matched against WORKSPACE_PACKAGES
 *    to detect monorepo cross-module imports through unlisted subpaths
 */
function resolvesToModule(importPath: string, consumerFile: string): string | null {
  // Case 2: Bare specifier — check if it targets this module's workspace package
  if (!importPath.startsWith('.')) {
    // WORKSPACE_PACKAGES maps package names to their directories.
    // If the import matches a package that maps to MODULE_PATH,
    // check whether the subpath goes through the barrel or bypasses it.
    for (const [pkgName, pkgDir] of Object.entries(WORKSPACE_PACKAGES)) {
      if (importPath === pkgName || importPath.startsWith(pkgName + '/')) {
        const normalizedPkgDir = path.resolve(pkgDir);
        if (normalizedPkgDir !== path.resolve(MODULE_PATH)) continue;
        // Import targets this module. Check if it's the barrel or a subpath.
        if (importPath === pkgName) return null; // Barrel import — OK
        // Subpath import — check if listed in package.json exports
        // (The orchestrator should only include non-exports subpaths as violations;
        //  this template assumes the orchestrator filtered exports-listed subpaths.)
        const subpath = importPath.slice(pkgName.length);
        const resolved = path.resolve(pkgDir, '.' + subpath);
        if (resolved.startsWith(normalizedPkgDir + path.sep)) return resolved;
      }
    }
    return null; // Not targeting this module
  }

  // Case 1: Relative import

  const consumerDir = path.dirname(path.resolve(consumerFile));
  let resolved = path.resolve(consumerDir, importPath);

  // Normalize: if the resolved path has no extension, try each source extension
  if (!path.extname(resolved)) {
    for (const ext of SOURCE_EXTENSIONS) {
      const withExt = resolved + ext;
      if (fs.existsSync(withExt)) { resolved = withExt; break; }
    }
    // Also try /index with each extension
    for (const ext of SOURCE_EXTENSIONS) {
      const indexPath = path.join(resolved, 'index' + ext);
      if (fs.existsSync(indexPath)) { resolved = indexPath; break; }
    }
  }

  const moduleDirResolved = path.resolve(MODULE_PATH);
  if (resolved.startsWith(moduleDirResolved + path.sep) || resolved === moduleDirResolved) {
    return resolved;
  }
  return null;
}

/**
 * Check whether a resolved path points to the barrel file.
 */
function isBarrelPath(resolvedPath: string): boolean {
  const barrelResolved = path.resolve(BARREL_PATH);
  // Direct match
  if (resolvedPath === barrelResolved) return true;
  // Match without extension (import might omit .ts/.js)
  const barrelNoExt = barrelResolved.replace(/\.[^.]+$/, '');
  if (resolvedPath === barrelNoExt) return true;
  // Match when importing the directory (resolves to index file)
  if (resolvedPath === path.dirname(barrelResolved)
      && path.basename(barrelResolved).startsWith('index')) return true;
  return false;
}

describe(`API contract: ${MODULE_NAME}`, () => {
  const consumerFiles = getConsumerFiles();

  test('all imports targeting the module go through the barrel file', () => {
    const violations: string[] = [];

    for (const file of consumerFiles) {
      const imports = extractImports(file);
      for (const imp of imports) {
        const resolved = resolvesToModule(imp.importPath, file);
        if (resolved === null) continue;
        if (!isBarrelPath(resolved)) {
          violations.push(
            `${file}:${imp.line} imports from ${path.relative('.', resolved)} — should import from ${BARREL_PATH}`
          );
        }
      }
    }

    expect(violations).toEqual([]);
  });
});
```

## Python (pytest) Template

File: `tests/structural/api-contracts-{MODULE_NAME}.test.py`

```python
import re
from pathlib import Path

# --- Generated by api-contract-guard: {MODULE_NAME} ---

MODULE_NAME = '{MODULE_NAME}'
MODULE_PATH = Path('{MODULE_PATH}')
BARREL_PATH = Path('{BARREL_PATH}')
SOURCE_EXTENSIONS = [{SOURCE_EXTENSIONS}]


def _get_consumer_files() -> list[Path]:
    """Collect all source files in the project OUTSIDE the guarded module directory."""
    project_root = Path('.')
    files: list[Path] = []
    module_resolved = MODULE_PATH.resolve()
    for ext in SOURCE_EXTENSIONS:
        for f in project_root.rglob(f'*{ext}'):
            resolved = f.resolve()
            # Skip files inside the guarded module
            if _is_inside(resolved, module_resolved):
                continue
            # Skip common non-project dirs
            parts = f.parts
            if any(p in ('.venv', 'venv', '__pycache__', '.tox', 'node_modules') for p in parts):
                continue
            files.append(f)
    return files


def _is_inside(child: Path, parent: Path) -> bool:
    """Check whether child path is inside parent directory."""
    try:
        child.relative_to(parent)
        return True
    except ValueError:
        return False


def _extract_imports(filepath: Path) -> list[dict]:
    """Extract Python import statements from a file.

    Returns list of {line, module_path} dicts for:
      - from X.Y import Z
      - import X.Y
    """
    text = filepath.read_text(encoding='utf-8')
    results = []

    for i, line_text in enumerate(text.splitlines(), start=1):
        stripped = line_text.strip()
        if stripped.startswith('#'):
            continue

        # from X.Y.Z import ...
        from_match = re.match(r'^from\s+([\w.]+)\s+import\s+', stripped)
        if from_match:
            results.append({'line': i, 'module_path': from_match.group(1)})
            continue

        # import X.Y.Z  (possibly with commas: import X.Y, A.B)
        import_match = re.match(r'^import\s+([\w.,\s]+)', stripped)
        if import_match:
            for mod in import_match.group(1).split(','):
                mod = mod.strip().split(' as ')[0].strip()
                if mod:
                    results.append({'line': i, 'module_path': mod})

    return results


def _module_path_to_fs(module_path_str: str) -> Path | None:
    """Convert a dotted Python import path to a filesystem path.

    Returns the path if it resolves to a location inside the guarded module,
    otherwise None.
    """
    parts = module_path_str.split('.')
    module_resolved = MODULE_PATH.resolve()

    # Try as package directory: X/Y/Z/__init__.py
    candidate_dir = Path(*parts)
    if candidate_dir.resolve() == module_resolved or _is_inside(candidate_dir.resolve(), module_resolved):
        return candidate_dir.resolve()

    # Try as module file: X/Y/Z.py
    candidate_file = Path(*parts[:-1]) / (parts[-1] + '.py') if len(parts) > 1 else Path(parts[0] + '.py')
    if _is_inside(candidate_file.resolve(), module_resolved):
        return candidate_file.resolve()

    # Try as package init: X/Y/Z/__init__.py
    candidate_init = Path(*parts) / '__init__.py'
    if _is_inside(candidate_init.resolve(), module_resolved):
        return candidate_init.resolve()

    return None


def _is_barrel_import(resolved_path: Path) -> bool:
    """Check whether the resolved import target is the barrel file (__init__.py)."""
    barrel_resolved = BARREL_PATH.resolve()

    # Direct match to __init__.py
    if resolved_path == barrel_resolved:
        return True

    # Import of the package directory itself (resolves to __init__.py)
    if resolved_path == barrel_resolved.parent and barrel_resolved.name == '__init__.py':
        return True

    return False


def test_all_imports_go_through_barrel():
    """All imports targeting {MODULE_NAME} must go through the barrel file ({BARREL_PATH})."""
    consumer_files = _get_consumer_files()
    violations = []

    for f in consumer_files:
        for imp in _extract_imports(f):
            resolved = _module_path_to_fs(imp['module_path'])
            if resolved is None:
                continue
            if not _is_barrel_import(resolved):
                rel = resolved.relative_to(Path('.').resolve()) if _is_inside(resolved, Path('.').resolve()) else resolved
                violations.append(
                    f"{f}:{imp['line']} imports from {rel} — should import from {BARREL_PATH}"
                )

    assert not violations, (
        f"Barrel bypass violations for module '{MODULE_NAME}':\n" + "\n".join(violations)
    )
```

## .NET (xUnit) Template

File: `Tests/Structural/ApiContracts{MODULE_PASCAL}Tests.cs`

```csharp
using System;
using System.Collections.Generic;
using System.IO;
using System.Linq;
using System.Text.RegularExpressions;
using Xunit;

// --- Generated by api-contract-guard: {MODULE_NAME} ---

public class ApiContracts{MODULE_PASCAL}Tests
{
    private const string ModuleName = "{MODULE_NAME}";
    private const string ModulePath = "{MODULE_PATH}";
    private const string BarrelNamespace = "{BARREL_NAMESPACE}";
    private static readonly string[] SourceExtensions = new[] { {SOURCE_EXTENSIONS} };
    private static readonly string[] InternalNamespaces = new[] { {MODULE_INTERNAL_NAMESPACES} };

    /// <summary>
    /// Collect all source files in the project OUTSIDE the guarded module directory.
    /// </summary>
    private static IEnumerable<string> GetConsumerFiles()
    {
        var projectRoot = Directory.GetCurrentDirectory();
        var moduleResolved = Path.GetFullPath(ModulePath);

        return SourceExtensions.SelectMany(ext =>
                Directory.EnumerateFiles(projectRoot, $"*{ext}", SearchOption.AllDirectories))
            .Where(f => !Path.GetFullPath(f).StartsWith(moduleResolved + Path.DirectorySeparatorChar)
                        && Path.GetFullPath(f) != moduleResolved)
            .Where(f => !f.Contains($"{Path.DirectorySeparatorChar}bin{Path.DirectorySeparatorChar}")
                        && !f.Contains($"{Path.DirectorySeparatorChar}obj{Path.DirectorySeparatorChar}"));
    }

    /// <summary>
    /// Extract using statements from a C# source file.
    /// Returns (lineNumber, namespace) tuples.
    /// </summary>
    private static IEnumerable<(int Line, string Namespace)> ExtractUsings(string filePath)
    {
        var lines = File.ReadAllLines(filePath);
        var usingPattern = new Regex(@"^\s*using\s+(?:static\s+)?([A-Za-z][\w.]*)\s*;");

        for (int i = 0; i < lines.Length; i++)
        {
            var match = usingPattern.Match(lines[i]);
            if (match.Success)
            {
                yield return (i + 1, match.Groups[1].Value);
            }
        }
    }

    /// <summary>
    /// Check whether a namespace references an internal part of the guarded module
    /// (not the barrel namespace).
    /// </summary>
    private static bool IsInternalNamespaceReference(string ns)
    {
        // If it exactly matches or starts with an internal namespace, it's a bypass
        return InternalNamespaces.Any(intern =>
            ns.Equals(intern, StringComparison.Ordinal)
            || ns.StartsWith(intern + ".", StringComparison.Ordinal));
    }

    /// <summary>
    /// Check whether a namespace references the barrel (public API) namespace.
    /// </summary>
    private static bool IsBarrelReference(string ns)
    {
        return ns.Equals(BarrelNamespace, StringComparison.Ordinal)
               || ns.StartsWith(BarrelNamespace + ".", StringComparison.Ordinal);
    }

    /// <summary>
    /// Check whether a namespace belongs to the guarded module at all.
    /// </summary>
    private static bool IsModuleNamespace(string ns)
    {
        return IsBarrelReference(ns) || IsInternalNamespaceReference(ns);
    }

    [Fact]
    public void AllUsingsTargetingModuleGoThroughBarrelNamespace()
    {
        var violations = new List<string>();

        foreach (var file in GetConsumerFiles())
        {
            foreach (var (line, ns) in ExtractUsings(file))
            {
                if (!IsModuleNamespace(ns))
                    continue;

                if (!IsBarrelReference(ns))
                {
                    var relPath = Path.GetRelativePath(Directory.GetCurrentDirectory(), file);
                    violations.Add(
                        $"{relPath}:{line} imports from {ns} — should import from {BarrelNamespace}");
                }
            }
        }

        Assert.True(violations.Count == 0,
            $"Barrel bypass violations for module '{ModuleName}':\n" + string.Join("\n", violations));
    }
}
```

### .NET Placeholder Notes

The .NET template uses two additional placeholders beyond the common set:

| Placeholder | Source | Example |
|---|---|---|
| `{MODULE_PASCAL}` | PascalCase module name | `Auth` |
| `{BARREL_NAMESPACE}` | Public barrel namespace | `MyApp.Auth` |
| `{MODULE_INTERNAL_NAMESPACES}` | Internal sub-namespaces | `["MyApp.Auth.Strategies", "MyApp.Auth.Guards"]` |

These are derived during module analysis: `{MODULE_PASCAL}` is the PascalCase form
of `{MODULE_NAME}`, and namespace placeholders map to the .NET namespace conventions
corresponding to `{BARREL_PATH}` and `{MODULE_INTERNAL_DIRS}`.
