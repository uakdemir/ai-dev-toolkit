# Convention Test Templates

Structural test code templates for convention categories routed to `structural_test`
or `both` (hybrid). Loaded at Step 9 (Generate Artifacts).

Each template is a complete, runnable test file for its target stack. Placeholders
are substituted by the orchestrator using confirmed convention data from Step 6 and
agent output:

- `{CONVENTION_DESCRIPTION}` — human-readable convention name from the finding
- `{EXPECTED_PATTERN}` — regex or string the conforming code matches
- `{BASE_ERROR_CLASS}` — detected base error/exception class name
- `{SOURCE_DIRS}` — directories to scan (from agent `files_analyzed` paths)
- `{SOURCE_EXTENSIONS}` — file extensions for the detected stack
- `{ENVELOPE_FIELDS}` — expected response envelope field names
- `{ENVELOPE_TYPE}` — type name or shape of the expected response envelope
- `{DIR_ROLE_MAP}` — mapping of directories to their expected file roles
- `{DI_CONTAINER_IMPORT}` — import path for the DI container/registration module
- `{DI_REGISTRATION_PATTERN}` — regex matching valid DI registration calls
- `{DI_REGISTRATION_FILES}` — glob patterns or paths for DI registration files
- `{SERVICE_CLASS_PATTERN}` — regex matching service class names
- `{CATEGORY}` — the convention category slug (e.g., `error-handling`)
- `{CATEGORY_SLUG}` — snake_case version for Python identifiers
- `{CATEGORY_PASCAL}` — PascalCase version for .NET class names
- `{DIRECTORY_ROLE_MAP}` — full directory-to-role mapping object literal

Categories that produce structural tests:
- **Error Handling** (hybrid) — structural portion checks class hierarchy
- **DI / Dependency Patterns** — structural test
- **API Response Shapes** — structural test
- **File Organization** — structural test
- **Discovery findings** — structural test (generic template)

Categories that produce linter rules only (no templates here):
- Naming Conventions, Import Ordering, Logging

## Error Handling Templates (structural portion of hybrid)

The linter portion of Error Handling (catch patterns, bare throws) is handled by
`linter-rule-mappings.md`. These structural tests cover the class hierarchy: all
custom errors must extend the detected base class, and no raw/untyped errors should
be thrown where a custom hierarchy exists.

### Node.js (Jest/Vitest)

File: `tests/structural/convention-error-handling.test.ts`

```typescript
import * as fs from 'fs';
import * as path from 'path';
import { glob } from 'glob';

// --- Generated by convention-enforcer: error-handling ---

const SOURCE_DIRS = [{SOURCE_DIRS}];
const SOURCE_EXTENSIONS = [{SOURCE_EXTENSIONS}];
const BASE_ERROR_CLASS = '{BASE_ERROR_CLASS}';

function getSourceFiles(): string[] {
  const patterns = SOURCE_DIRS.flatMap(dir =>
    SOURCE_EXTENSIONS.map(ext => `${dir}/**/*${ext}`)
  );
  return patterns.flatMap(p => glob.sync(p));
}

function extractClassHierarchy(filePath: string): Array<{ className: string; extendsFrom: string | null }> {
  const src = fs.readFileSync(filePath, 'utf-8');
  const classPattern = /class\s+(\w+)\s+extends\s+(\w+)/g;
  const results: Array<{ className: string; extendsFrom: string | null }> = [];
  let match: RegExpExecArray | null;
  while ((match = classPattern.exec(src)) !== null) {
    results.push({ className: match[1], extendsFrom: match[2] });
  }
  return results;
}

function findRawThrows(filePath: string): Array<{ line: number; text: string }> {
  const src = fs.readFileSync(filePath, 'utf-8');
  const lines = src.split('\n');
  const violations: Array<{ line: number; text: string }> = [];
  for (let i = 0; i < lines.length; i++) {
    const line = lines[i];
    // Match `throw new Error(` but not `throw new SomethingError(` where
    // SomethingError is not the bare Error class.
    if (/throw\s+new\s+Error\s*\(/.test(line)) {
      violations.push({ line: i + 1, text: line.trim() });
    }
  }
  return violations;
}

function findErrorClasses(filePath: string): string[] {
  const src = fs.readFileSync(filePath, 'utf-8');
  const pattern = /class\s+(\w*(?:Error|Exception)\w*)/g;
  const names: string[] = [];
  let match: RegExpExecArray | null;
  while ((match = pattern.exec(src)) !== null) {
    names.push(match[1]);
  }
  return names;
}

describe('{CONVENTION_DESCRIPTION}', () => {
  const files = getSourceFiles();

  test('all custom error classes extend the base error class', () => {
    const violations: string[] = [];
    for (const file of files) {
      const hierarchy = extractClassHierarchy(file);
      const errorClasses = findErrorClasses(file);
      for (const errorClass of errorClasses) {
        if (errorClass === BASE_ERROR_CLASS) continue;
        const entry = hierarchy.find(h => h.className === errorClass);
        if (entry && entry.extendsFrom !== BASE_ERROR_CLASS) {
          // Check for indirect inheritance: if extends another custom error,
          // that's acceptable as long as the chain reaches BASE_ERROR_CLASS.
          // This single-file check only verifies the direct parent — deeper
          // chain validation requires cross-file resolution.
          violations.push(
            `${file}: ${errorClass} extends ${entry.extendsFrom}, expected ${BASE_ERROR_CLASS}`
          );
        }
      }
    }
    expect(violations).toEqual([]);
  });

  test('no raw Error throws when custom error hierarchy exists', () => {
    const violations: string[] = [];
    for (const file of files) {
      const rawThrows = findRawThrows(file);
      for (const v of rawThrows) {
        violations.push(`${file}:${v.line}: ${v.text}`);
      }
    }
    expect(violations).toEqual([]);
  });
});
```

```python
import ast
import os
from pathlib import Path

# --- Generated by convention-enforcer: error-handling ---

SOURCE_DIRS = [{SOURCE_DIRS}]
SOURCE_EXTENSIONS = [{SOURCE_EXTENSIONS}]
BASE_ERROR_CLASS = '{BASE_ERROR_CLASS}'


def _get_source_files() -> list[Path]:
    files: list[Path] = []
    for d in SOURCE_DIRS:
        root = Path(d)
        if root.exists():
            for ext in SOURCE_EXTENSIONS:
                files.extend(root.rglob(f'*{ext}'))
    return files


def _extract_class_hierarchy(filepath: Path) -> list[dict]:
    """Return list of {name, bases} for all class definitions in the file."""
    tree = ast.parse(filepath.read_text(encoding='utf-8'), filename=str(filepath))
    results = []
    for node in ast.walk(tree):
        if isinstance(node, ast.ClassDef):
            bases = []
            for base in node.bases:
                if isinstance(base, ast.Name):
                    bases.append(base.id)
                elif isinstance(base, ast.Attribute):
                    bases.append(base.attr)
            results.append({'name': node.name, 'bases': bases})
    return results


def _find_bare_raises(filepath: Path) -> list[dict]:
    """Find bare `raise Exception(...)` or `raise BaseException(...)` calls."""
    tree = ast.parse(filepath.read_text(encoding='utf-8'), filename=str(filepath))
    violations = []
    for node in ast.walk(tree):
        if isinstance(node, ast.Raise) and node.exc is not None:
            exc = node.exc
            if isinstance(exc, ast.Call) and isinstance(exc.func, ast.Name):
                if exc.func.id in ('Exception', 'BaseException'):
                    violations.append({'line': node.lineno, 'text': f'raise {exc.func.id}(...)'})
    return violations


def _is_error_class(name: str) -> bool:
    return name.endswith('Error') or name.endswith('Exception')


def test_all_custom_errors_extend_base_class():
    """All custom error/exception classes must extend {BASE_ERROR_CLASS}."""
    files = _get_source_files()
    violations = []
    for f in files:
        for cls in _extract_class_hierarchy(f):
            if not _is_error_class(cls['name']):
                continue
            if cls['name'] == BASE_ERROR_CLASS:
                continue
            if BASE_ERROR_CLASS not in cls['bases']:
                # Single-file check: verifies direct parent only. Indirect
                # inheritance through another custom error is acceptable but
                # requires cross-file resolution to validate fully.
                violations.append(
                    f"{f}: {cls['name']} extends {cls['bases']}, "
                    f"expected {BASE_ERROR_CLASS}"
                )
    assert not violations, (
        "Custom error classes not extending base class:\n" + "\n".join(violations)
    )


def test_no_bare_exception_raises():
    """No bare raise Exception() when custom error hierarchy exists."""
    files = _get_source_files()
    violations = []
    for f in files:
        for v in _find_bare_raises(f):
            violations.append(f"{f}:{v['line']}: {v['text']}")
    assert not violations, (
        "Bare Exception raises found:\n" + "\n".join(violations)
    )
```

## DI / Dependency Patterns Templates

These structural tests verify that dependencies are registered through the DI
container and that service classes are not directly instantiated outside of DI
registration files.

### Node.js (Jest/Vitest)

File: `tests/structural/convention-di-patterns.test.ts`

```typescript
import * as fs from 'fs';
import * as path from 'path';
import { glob } from 'glob';

// --- Generated by convention-enforcer: di-patterns ---

const SOURCE_DIRS = [{SOURCE_DIRS}];
const SOURCE_EXTENSIONS = [{SOURCE_EXTENSIONS}];
const DI_REGISTRATION_FILES = [{DI_REGISTRATION_FILES}];
const SERVICE_CLASS_PATTERN = /{SERVICE_CLASS_PATTERN}/;
const DI_REGISTRATION_PATTERN = /{DI_REGISTRATION_PATTERN}/;

function getSourceFiles(): string[] {
  const patterns = SOURCE_DIRS.flatMap(dir =>
    SOURCE_EXTENSIONS.map(ext => `${dir}/**/*${ext}`)
  );
  return patterns.flatMap(p => glob.sync(p));
}

function isDIRegistrationFile(filePath: string): boolean {
  return DI_REGISTRATION_FILES.some(pattern => filePath.includes(pattern));
}

function findDirectInstantiations(filePath: string): Array<{ line: number; text: string; className: string }> {
  const src = fs.readFileSync(filePath, 'utf-8');
  const lines = src.split('\n');
  const violations: Array<{ line: number; text: string; className: string }> = [];
  for (let i = 0; i < lines.length; i++) {
    const line = lines[i];
    // Match `new ServiceName(` where ServiceName matches the service class pattern.
    const match = line.match(/new\s+(\w+)\s*\(/);
    if (match && SERVICE_CLASS_PATTERN.test(match[1])) {
      violations.push({ line: i + 1, text: line.trim(), className: match[1] });
    }
  }
  return violations;
}

function findHardcodedImports(filePath: string): Array<{ line: number; text: string }> {
  const src = fs.readFileSync(filePath, 'utf-8');
  const lines = src.split('\n');
  const violations: Array<{ line: number; text: string }> = [];
  for (let i = 0; i < lines.length; i++) {
    const line = lines[i];
    // Detect reaching into internal service paths directly rather than via DI.
    // Matches imports that pull a concrete service class from a deep internal path.
    const importMatch = line.match(/(?:import|require)\s*\(?.*['"]([^'"]+)['"]/);
    if (importMatch) {
      const importPath = importMatch[1];
      // Flag imports that reach into another service's internal directory.
      // e.g., importing from '../services/auth/internal/tokenStore'
      if (/\/internal\//.test(importPath)) {
        violations.push({ line: i + 1, text: line.trim() });
      }
    }
  }
  return violations;
}

describe('{CONVENTION_DESCRIPTION}', () => {
  const files = getSourceFiles();

  test('service classes are not directly instantiated outside DI registration', () => {
    const violations: string[] = [];
    for (const file of files) {
      if (isDIRegistrationFile(file)) continue;
      // Also skip test files — tests may legitimately instantiate services.
      if (/\.(test|spec)\.[jt]sx?$/.test(file)) continue;
      const directInstantiations = findDirectInstantiations(file);
      for (const v of directInstantiations) {
        violations.push(`${file}:${v.line}: direct instantiation of ${v.className} — ${v.text}`);
      }
    }
    expect(violations).toEqual([]);
  });

  test('DI registration files use the DI container pattern', () => {
    const violations: string[] = [];
    for (const file of files) {
      if (!isDIRegistrationFile(file)) continue;
      const src = fs.readFileSync(file, 'utf-8');
      // Verify that DI registration files actually contain registration calls.
      if (!DI_REGISTRATION_PATTERN.test(src)) {
        violations.push(`${file}: DI registration file does not contain expected registration pattern`);
      }
    }
    expect(violations).toEqual([]);
  });

  test('no hardcoded internal path imports', () => {
    const violations: string[] = [];
    for (const file of files) {
      if (isDIRegistrationFile(file)) continue;
      if (/\.(test|spec)\.[jt]sx?$/.test(file)) continue;
      const hardcoded = findHardcodedImports(file);
      for (const v of hardcoded) {
        violations.push(`${file}:${v.line}: ${v.text}`);
      }
    }
    expect(violations).toEqual([]);
  });
});
```

```python
import ast
import os
import re
from pathlib import Path

# --- Generated by convention-enforcer: di-patterns ---

SOURCE_DIRS = [{SOURCE_DIRS}]
SOURCE_EXTENSIONS = [{SOURCE_EXTENSIONS}]
DI_REGISTRATION_FILES = [{DI_REGISTRATION_FILES}]
SERVICE_CLASS_PATTERN = re.compile(r'{SERVICE_CLASS_PATTERN}')
DI_REGISTRATION_PATTERN = re.compile(r'{DI_REGISTRATION_PATTERN}')


def _get_source_files() -> list[Path]:
    files: list[Path] = []
    for d in SOURCE_DIRS:
        root = Path(d)
        if root.exists():
            for ext in SOURCE_EXTENSIONS:
                files.extend(root.rglob(f'*{ext}'))
    return files


def _is_di_registration_file(filepath: Path) -> bool:
    return any(pattern in str(filepath) for pattern in DI_REGISTRATION_FILES)


def _is_test_file(filepath: Path) -> bool:
    name = filepath.name
    return name.startswith('test_') or name.endswith('_test.py') or '/tests/' in str(filepath)


def _find_direct_instantiations(filepath: Path) -> list[dict]:
    """Find `ClassName(...)` calls where ClassName matches service class pattern."""
    tree = ast.parse(filepath.read_text(encoding='utf-8'), filename=str(filepath))
    violations = []
    for node in ast.walk(tree):
        if isinstance(node, ast.Call) and isinstance(node.func, ast.Name):
            if SERVICE_CLASS_PATTERN.match(node.func.id):
                violations.append({
                    'line': node.lineno,
                    'class_name': node.func.id,
                })
    return violations


def _find_internal_imports(filepath: Path) -> list[dict]:
    """Find imports that reach into another module's internal directory."""
    tree = ast.parse(filepath.read_text(encoding='utf-8'), filename=str(filepath))
    violations = []
    for node in ast.walk(tree):
        if isinstance(node, ast.ImportFrom) and node.module:
            if '.internal.' in node.module or node.module.endswith('.internal'):
                violations.append({
                    'line': node.lineno,
                    'module': node.module,
                })
    return violations


def test_no_direct_service_instantiation():
    """Service classes must not be instantiated outside DI registration files."""
    files = _get_source_files()
    violations = []
    for f in files:
        if _is_di_registration_file(f) or _is_test_file(f):
            continue
        for v in _find_direct_instantiations(f):
            violations.append(
                f"{f}:{v['line']}: direct instantiation of {v['class_name']}"
            )
    assert not violations, (
        "Direct service instantiations found outside DI registration:\n"
        + "\n".join(violations)
    )


def test_di_registration_files_use_container():
    """DI registration files must contain expected registration pattern."""
    files = _get_source_files()
    violations = []
    for f in files:
        if not _is_di_registration_file(f):
            continue
        src = f.read_text(encoding='utf-8')
        if not DI_REGISTRATION_PATTERN.search(src):
            violations.append(
                f"{f}: DI registration file does not contain expected registration pattern"
            )
    assert not violations, (
        "DI registration files missing container pattern:\n"
        + "\n".join(violations)
    )


def test_no_internal_path_imports():
    """No imports reaching into another module's internal directory."""
    files = _get_source_files()
    violations = []
    for f in files:
        if _is_di_registration_file(f) or _is_test_file(f):
            continue
        for v in _find_internal_imports(f):
            violations.append(f"{f}:{v['line']}: import from {v['module']}")
    assert not violations, (
        "Internal path imports found:\n" + "\n".join(violations)
    )
```

## API Response Shapes Templates

These structural tests verify that all endpoint/handler files return responses
conforming to the detected envelope structure.

### Node.js (Jest/Vitest)

File: `tests/structural/convention-api-response.test.ts`

```typescript
import * as fs from 'fs';
import * as path from 'path';
import { glob } from 'glob';

// --- Generated by convention-enforcer: api-response ---

const ENDPOINT_DIRS = [{SOURCE_DIRS}];
const SOURCE_EXTENSIONS = [{SOURCE_EXTENSIONS}];
const ENVELOPE_FIELDS = [{ENVELOPE_FIELDS}];

function getEndpointFiles(): string[] {
  const patterns = ENDPOINT_DIRS.flatMap(dir =>
    SOURCE_EXTENSIONS.map(ext => `${dir}/**/*${ext}`)
  );
  return patterns.flatMap(p => glob.sync(p));
}

function extractReturnStatements(filePath: string): Array<{ line: number; text: string }> {
  const src = fs.readFileSync(filePath, 'utf-8');
  const lines = src.split('\n');
  const results: Array<{ line: number; text: string }> = [];
  for (let i = 0; i < lines.length; i++) {
    const line = lines[i];
    // Match reply/send/json/return calls that construct a response object.
    // Covers Fastify: reply.send({...}), Express: res.json({...}),
    // and direct return statements in route handlers.
    if (/(?:reply\.send|res\.json|res\.send|return\s*)\s*\(\s*\{/.test(line)) {
      // Capture the response object — may span multiple lines.
      // Extract from the opening brace to find the keys at the top level.
      const remaining = lines.slice(i).join('\n');
      const braceStart = remaining.indexOf('{');
      if (braceStart !== -1) {
        results.push({ line: i + 1, text: remaining.substring(braceStart, braceStart + 200) });
      }
    }
  }
  return results;
}

function responseHasEnvelopeFields(responseText: string): { missing: string[]; present: string[] } {
  const missing: string[] = [];
  const present: string[] = [];
  for (const field of ENVELOPE_FIELDS) {
    // Check for the field as an object key: `fieldName:` or `"fieldName":` or `'fieldName':`
    const fieldPattern = new RegExp(`(?:['"]?${field}['"]?)\\s*:`);
    if (fieldPattern.test(responseText)) {
      present.push(field);
    } else {
      missing.push(field);
    }
  }
  return { missing, present };
}

describe('{CONVENTION_DESCRIPTION}', () => {
  const files = getEndpointFiles();

  test('all endpoint responses use the standard envelope shape', () => {
    const violations: string[] = [];
    for (const file of files) {
      const returns = extractReturnStatements(file);
      for (const ret of returns) {
        const { missing } = responseHasEnvelopeFields(ret.text);
        if (missing.length > 0) {
          violations.push(
            `${file}:${ret.line}: response missing envelope fields: ${missing.join(', ')}`
          );
        }
      }
    }
    expect(violations).toEqual([]);
  });
});
```

```python
def _extract_return_dicts(filepath: Path) -> list[dict]:
    """Find return statements that return a dict literal in route handler functions."""
    tree = ast.parse(filepath.read_text(encoding='utf-8'), filename=str(filepath))
    results = []
    for node in ast.walk(tree):
        if isinstance(node, ast.Return) and node.value is not None:
            # Direct dict return: return {"data": ..., "status": ...}
            if isinstance(node.value, ast.Dict):
                keys = []
                for key in node.value.keys:
                    if isinstance(key, ast.Constant) and isinstance(key.value, str):
                        keys.append(key.value)
                results.append({'line': node.lineno, 'keys': keys})
            # Dict constructor in a response wrapper: return JSONResponse({"data": ...})
            # or return Response(content={...})
            elif isinstance(node.value, ast.Call):
                for arg in node.value.args:
                    if isinstance(arg, ast.Dict):
                        keys = []
                        for key in arg.keys:
                            if isinstance(key, ast.Constant) and isinstance(key.value, str):
                                keys.append(key.value)
                        results.append({'line': node.lineno, 'keys': keys})
                for kw in node.value.keywords:
                    if isinstance(kw.value, ast.Dict):
                        keys = []
                        for key in kw.value.keys:
                            if isinstance(key, ast.Constant) and isinstance(key.value, str):
                                keys.append(key.value)
                        results.append({'line': node.lineno, 'keys': keys})
    return results


def test_all_endpoints_use_standard_envelope():
    """All endpoint responses must include the standard envelope fields."""
    files = _get_endpoint_files()
    violations = []
    for f in files:
        for ret in _extract_return_dicts(f):
            missing = [field for field in ENVELOPE_FIELDS if field not in ret['keys']]
            if missing:
                violations.append(
                    f"{f}:{ret['line']}: response missing envelope fields: {', '.join(missing)}"
                )
    assert not violations, (
        "Endpoints not using standard envelope shape:\n" + "\n".join(violations)
    )
```

## Conflict Resolution

When existing test files are detected at target paths:

**Generate separately** — write to alternate filenames:

| Stack   | Alternate filename pattern |
|---|---|
| Node.js | `convention-{category}-generated.test.ts` |
| Python  | `convention-{category}-generated.test.py` |
| .NET    | `Convention{Category}Tests.Generated.cs` |

**Merge mode** — append after the existing marker comment in the file. The marker
comment is the dedup key:

- JS/TS/C#: `// --- Generated by convention-enforcer: {category} ---`
- Python: `# --- Generated by convention-enforcer: {category} ---`

If a file already contains a marker for the same category, **replace** the block
between that marker and the next marker (or end of file). This makes re-runs
idempotent.

## Template Selection Logic

The orchestrator selects templates based on:

1. **Category** — determines which template section to use
2. **Stack** — determines which language template within the section
3. **Routing** — only categories routed to `structural_test` or `both` get templates

| Category | Routing | Template section |
|---|---|---|
| Error Handling | `both` (hybrid) | Error Handling Templates |
| DI / Dependency Patterns | `structural_test` | DI / Dependency Patterns Templates |
| API Response Shapes | `structural_test` | API Response Shapes Templates |
| File Organization | `structural_test` | File Organization Templates |
| Naming Conventions | `linter_rule` | No template (linter only) |
| Import Ordering | `linter_rule` | No template (linter only) |
| Logging | `linter_rule` | No template (linter only) |
| Discovery findings | `structural_test` (default) | Discovery Finding Templates |

If a user overrides routing at Step 8 (e.g., changes DI from `structural_test` to
`linter_rule`), skip the structural test template for that category. If a user
overrides a linter-only category to `structural_test`, use the Discovery Finding
generic template with the category's convention description and pattern.
